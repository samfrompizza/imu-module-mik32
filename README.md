# MIK32 IMU Level & Compass

Учебный pet-проект для платы **ELBEAR / MIK32 RISC-V**: электронный уровень и компас на основе датчика **BNO055** и **OLED 1.3"**.

## Что делает проект

* показывает наклон платы в режиме уровня;
* показывает направление в режиме компаса;
* режимы переключаются кнопкой;
* читает BNO055 по **I2C**;
* выводит изображение на OLED по **SPI**;
* использует таймерные прерывания для опроса датчика и обновления экрана.

## Железо

* MIK32 / ELBEAR
* BNO055 I2C
* OLED 0.96" SPI (MSP096X)
* 1 кнопка
* провода / макетная плата

## Зачем проект

Проект показывает работу с:

* I2C и SPI;
* прерываниями и таймерами;
* датчиками движения;
* графическим интерфейсом на микроконтроллере;
* простой событийной архитектурой;
* [OPT] в перспективе можно подключить DMA.

## Режимы

* **Level** — отображение наклона;
* **Compass** — отображение азимута;
* **Debug / Calibration** — служебный экран.
